## 정규 표현식 소개
* 정규 표현식을 사용하는 이유
    * 검색: 원하는 정보가 어디에 있는지 찾거나
    * 치환: 정보를 찾은 뒤에 편집하는 것

## 문자 하나 찾기
* `.` : 아무 문자 하나와 일치
* `\` : 메타 문자. 다른 메타 문자를 이스케이프 하기 위해 사용
    * 메타 문자: 문자 그대로 사용되지 않고 특별한 의미를 지니는 문자
    * 이스케이프: 메타 문자로서 지니는 의미 대신 문자 자체를 뜻하게 만드는 것
    * `\.` : 마침표

## 문자 집합으로 찾기
* `[]` : 집합
* `-` : 범위
    * `[]` 안에서만 메타문자
* `^` : 제외

## 메타 문자 사용하기
* 공백 메타 문자
    * `[\b]` : 역스페이스
    * `\f` : 페이지 넘김
    * `\n` : 줄바꿈
    * `\r` : 캐리지 리턴
    * `\t` : 탭
    * `\v` : 수직 탭
* 숫자 메타 문자
    * `\d` : 숫자 하나([0-9])
    * `\D` : 숫자 제외 문자 하나([^0-9])
* 영숫자 메타 문자
    * `\w` : 대소문자와 밑줄을 포함한 모든 영숫자([a-zA-Z0-9_])
    * `\W` : 영숫자, 밑줄 제외 모든 문자([^a-zA-Z0-9_])
